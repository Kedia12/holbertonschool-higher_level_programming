# Python - Hello World

This repository contains Python scripts from the Holberton School **Higher Level Programming** project `0x00-python-hello_world`.  
The purpose of this project is to learn **basic Python syntax, string manipulation, and the Zen of Python**.

## Requirements

- All files are interpreted/compiled on **Ubuntu 20.04 LTS** using **Python 3 (3.8.x)**.
- Scripts must be **executable** and start with the proper **shebang**:

```bash
#!/usr/bin/python3
| File                   | Description                                                     |
| ---------------------- | --------------------------------------------------------------- |
| `0-run`                | Shell script that runs `hello.py`                               |
| `1-run`                | Shell script that runs `hello_world.py`                         |
| `2-print.py`           | Prints `"Programming is like building a multilingual puzzle"`   |
| `3-print_number.py`    | Prints a number with a string using **f-strings**               |
| `4-print_float.py`     | Prints a float with **2 decimal places** using **f-strings**    |
| `4-print_float_pos.py` | Prints a positive float with **2 decimal places**               |
| `6-concat.py`          | Concatenates and prints two strings using **f-strings**         |
| `7-edges.py`           | Prints first 3 letters, last 2 letters, and middle of a word    |
| `9-easter_egg.py`      | Prints **The Zen of Python**, by Tim Peters (Python Easter egg) |
