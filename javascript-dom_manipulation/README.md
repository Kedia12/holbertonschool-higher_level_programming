JavaScript DOM Manipulation

Learning project for JavaScript DOM manipulation techniques.

Topics Covered

* Selecting DOM elements
* Modifying element properties and content
* Event handling
* Creating and removing elements
* Manipulating classes and styles

Usage

Open HTML files in a browser or run with a local server.
